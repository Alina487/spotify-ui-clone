# Spotify Web Player UI Clone 🎧

A pixel-perfect, static frontend UI clone of the Spotify web player interface. This project was built from scratch to practice structural layouts, semantic markup, and advanced CSS styling techniques without relying on external frameworks.

## 🚀 Live Demo
You can view the live project here: [👉 Click here to view Live Site](https://alina487.github.io/spotify-ui-clone/)

## 🛠️ Features Recreated
* *Sidebar Navigation:* Home, Search, Library buttons, and custom playlist creation sections.
* *Main Dashboard Layout:* Scrollable grid panels displaying featured albums, recent plays, and custom charts.
* *Sticky Media Controller:* Bottom playback bar featuring track details, playback control buttons (play, pause, skip), and volume sliders.
* *Hover Animations:* Realistic smooth scaling effects and color transitions on album cards and play buttons to mimic the native Spotify application.

## 💻 Tech Stack & Concepts Applied
* *HTML5:* Semantic architecture elements (<section>, <nav>, <aside>, <main>, <footer>) to ensure structural clarity.
* *CSS3 Custom Layouts:* Combination of *CSS Grid* for the multi-panel page layout and *Flexbox* for aligning buttons and alignment cards.
* *Responsive Architecture:* Customized media queries implemented to maintain layout integrity across desktop and laptop screens.
* *Vector Graphics:* Scalable SVG icon usage for crisp UI presentation.
